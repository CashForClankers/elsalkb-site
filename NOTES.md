# Build Notes

## Goal

Make the trip data visible in a beautiful, cheap, zero-backend website.

## Design choices

- Static site only
- Plain HTML/CSS/JS
- GitHub Pages deploy
- JSON exported from SQLite
- No signup, no server, no database exposed publicly

## Next useful upgrades

1. Search all 3477 experiences, not just featured rows
2. Map view with lightweight Leaflet layer
3. Detail pages per experience slug
4. Itinerary and booking status from the new itinerary table
5. Feedback-driven boosts from visited/skipped/loved-it signals
