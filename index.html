ElSalKB

Beautiful trip data, no backend.

A static, GitHub-hosted view over the El Salvador trip knowledge base. Fast, cheap, simple, and easy to evolve later into itinerary state, feedback, and contact enrichment.
—Total experiences
—Featured on site
—Areas
—Rows with contact info
Featured rows are sorted by blended average score for a clean first pass.
Built as a static export from SQLite → JSON → plain HTML/CSS/JS. Deploy target: GitHub Pages.
Notes live in NOTES.md and the repo README.
